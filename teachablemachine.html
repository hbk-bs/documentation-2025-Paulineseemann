Teachable Machine

Assignment

Create a project using Google’s Teachable Machine.

Realisation

For this project, I wanted to create an AI that can recognize chemical hazard symbols and, in a darkly humorous way, tell you what to beware of or what might happen if you decide to use the product. I collected many pictures of the different symbols and uploaded them into Teachable Machine. After training the model with this input, I embedded the final AI into my own website.

Images

Reflection

Sometimes the Teachable Machine AI struggles to recognize the images correctly, but I’m still happy that it recognizes anything at all. I think it would require a lot more images as input for it to work better, but for this project, it’s fine as it is, considering it’s meant to be a joke. I might even use it again to actually remember what a symbol means, since there are many dangerous substances at the HBK.
